Краткое руководство:
Запустите приложение
В меню нажмите: DB > Connection 
В появившемся окне выберите имя БД, ОК
В меню нажмите: * : Open table > Имя таблицы 
Можно открыть несколько таблиц
Таблицы можно редактировать. Для сохранения изменений в БД, нажмите (в окне таблицы) Save Change / ОК.
Закрытие окна таблицы не сохраняет внесенные изменения в БД
Для смены БД нажмите в меню: DB > Connection. 
Для удобства работы с таблицами, в меню есть настройки выбора и отображения окон, View.
